# 📍 How to Get the Repository URL on GitHub

## Step-by-Step Instructions

### 1. Go to the Repository Page
- After creating a new repository, GitHub will automatically take you to the repository page
- Or navigate to: `https://github.com/your-dad-username/repository-name`

### 2. Find the Green "Code" Button
- Look at the repository page
- You'll see a big **green "Code"** button near the top-right of the page
- It's usually next to "Add file" and "Go to file" buttons

### 3. Click the "Code" Button
- Click the green **"Code"** button
- A dropdown menu will appear

### 4. Copy the URL
- You'll see several tabs: "Local", "HTTPS", "SSH", "GitHub CLI"
- The **HTTPS** tab should be selected by default (highlighted)
- You'll see a URL that looks like: `https://github.com/username/repo-name.git`
- Click the **copy icon** (looks like two overlapping squares) next to the URL
- Or just select and copy the entire URL text

### 5. That's Your Repository URL!
- The URL should look like: `https://github.com/dad-username/repo-name.git`
- Or: `https://github.com/dad-username/repo-name` (both work the same)

## Quick Alternative

If the repository is already open, you can also:
- Look at your browser's address bar
- The URL in the address bar IS the repository URL!
- Just add `.git` at the end if it's not already there

Example:
- Address bar shows: `https://github.com/username/repo-name`
- Repository URL is: `https://github.com/username/repo-name.git`

---

## Still Can't Find It?

If you're having trouble:
1. Make sure you're signed into GitHub
2. Make sure the repository exists (was created successfully)
3. Try going directly to: `https://github.com/YOUR-DAD-USERNAME?tab=repositories`
4. Click on the repository name to open it
5. Then follow steps 2-4 above
