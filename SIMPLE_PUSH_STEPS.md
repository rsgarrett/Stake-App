# Simple Push Steps

Since origin doesn't exist, just continue with these commands:

## Run These Commands:

```bash
git remote add origin https://rsgarrett@github.com/rsgarrett/Stake-App.git
git branch -M main
git add .
git commit -m "Initial commit - Stake President App"
git push -u origin main
```

When prompted for password, paste the Personal Access Token.

---

## All in One Go:

Copy and paste these commands one at a time:

1. `git remote add origin https://rsgarrett@github.com/rsgarrett/Stake-App.git`
2. `git branch -M main`
