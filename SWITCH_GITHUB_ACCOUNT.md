# Switch GitHub Account in GitHub Desktop

## Steps to Switch to Your Dad's Account (rsgarrett):

1. **Sign Out of Current Account:**
   - In GitHub Desktop, click **File** → **Options** (or **Preferences** on Mac)
   - Go to the **Accounts** tab
   - Click **Sign Out** next to your current account (kinca)

2. **Sign In with Your Dad's Account:**
   - Click **Sign In** or **Sign in to GitHub.com**
   - You'll be taken to a browser
   - Sign in with your dad's GitHub account (rsgarrett)
   - Authorize GitHub Desktop

3. **Alternative: Add Account:**
   - You can also click **File** → **Options** → **Accounts**
   - Click **Add** or **Sign in**
   - Sign in with rsgarrett account
   - Then make it the default account

4. **Verify the Account:**
   - After signing in, check the top right corner of GitHub Desktop
   - It should show your dad's username (rsgarrett)
   - If it still shows "kinca", click on it and switch accounts

5. **Then Push:**
   - Once signed in as rsgarrett, try pushing again
   - It should now use the correct account

## Quick Alternative:

If you just want to push once, you can also:
1. In GitHub Desktop, go to **Repository** → **Repository Settings**
2. Change the remote URL to include the username:
   ```
   https://rsgarrett@github.com/rsgarrett/Stake-App.git
   ```
3. When you push, it will prompt for credentials
4. Use rsgarrett username and Personal Access Token

But signing in with the correct account is cleaner! ✅
